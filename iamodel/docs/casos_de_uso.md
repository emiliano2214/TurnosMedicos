# Casos de Uso Comunes

Preguntas y acciones frecuentes que los usuarios podrían consultar a la IA.

## Para Pacientes
- "¿Cuándo es mi próximo turno?"
- "¿Cómo cancelo un turno?"
- "¿Qué obra social aceptan?"
- "¿El Dr. Pérez atiende los viernes?"

## Para Médicos
- "¿Cuántos pacientes tengo hoy?"
- "¿Quién es el próximo paciente?"
- "Mostrame la historia clínica de Juan Gomez."
- "¿Cuántos turnos tuve esta semana?"

## Para Administrativos / Admin
- "¿Cuál es la especialidad con más demanda?"
- "¿Cuántos turnos se cancelaron ayer?"
- "Generar reporte de inasistencias del mes."
- "¿Qué médicos tienen la agenda vacía mañana?"
